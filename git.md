### GIT WORK FLOW :-

###### ================================================================================



1. mkdir directory name                      -->       to create a directory
2. git init                                                -->      to initialize the git
3. touch file name                                -->       to create file
4. git status                                          -->        to check the files wheather they are tracked or not
5. git add .                                            -->        if the files are  in un tracked the use this command to add the file in the present working directory
6. git remote add origin (GitHub url)-->       to link the local repo to remote repo
7. git config --global user.name "   "  -->       to give the user information to the repo
8. git config --global user.mail   "    " -->       to give the user mail info for the repo
9. git config  --list                                 -->       to see the info the repo
10. git commit -m "commit msg"        -->       to commit the file to the master branch
11. git push origin branch name          -->       to push the file form local to the remote
12. git pull origin branch name            -->       to pull the file from the remote to local

### BRANCH WORKFLOW:-

==============================================================================================================================



1. git branch branch name                  -->       to create a new branch
2. git checkout  branch name             -->       to change the branch
3. touch file name                                -->       to create file
4. git status                                          -->        to check the files wheather they are tracked or not
5. git add .                                            -->        if the files are  in un tracked the use this command to add the file in the present working directory
6. git commit -m "commit msg"        -->       to commit the file to the master branch
7. git push origin branch name          -->       to push the file form local to the remote
8. git pull origin branch name            -->       to pull the file from the remote to local

=============================================================================================================================

2. git checkout -b branch name          -->        to create and change the branch
